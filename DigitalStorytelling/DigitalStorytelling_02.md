# 디지털 스토리텔링 #2 - 예술사
## History
- Historea(라틴어원) : Hi(쓰다) + Sotrea(이야기)
- 역사는 이야기를 기록해 놓은 것이다. 우리는 역사를 통해 이야기를 통해 인간이 바라본 세계의 다양한 모습을 볼 수 있다.
## 서양 예술
- 육체(물질)와 영혼(정신)의 이분법적인 사고에서 출발.
- 예술은 인관과 사회, 인간과 자연, 인간과 역사와의 관계 속에서 육체(물질)와 영혼(정신)을 놓고 고민한다.
## 선사 예술
- 자연은 숭고한, 두려운 존재이다.
- 주술의 성격이 숭고한 존재가 됨
- 객관적 비례 : 자연주의적 묘사 - 사물과의 일치 중심 - Naturalism - 그리스 르네상스
- 제작적 비례 : 작가정신 이면, 관념, 사물과의 고려 없음. - Composition - 중세 예술, 현대 예술
- 예술은 존재하기 위해, 생존하기 위한 자연의 지혜를 터득하고 생리를 앎과 동시에 나눔에 의미가 있음
- 선사 시대와 역사 시대의 구분은 문자에 있다.
## 이집트/그리스 예술 (고전)
- 주술적 성격의 예술이 역사시대에 오면서 종교적 숭배의 대상이 되거나 미적 감상의 대상이 되었다.
- 이집트 벽화
    - 예술이 종교적 숭배의 대상. 종교적 기능에 종속
    - 언뜻 보면 제작적 비례(간단, 추상)
    - 인체의 비율도 지켜지고 있는 것을 보면 객관적 비례도 있음
    - 객관적 비례, 제작적 비례를 알맞게 사용함
    - why? 이들의 관심사는 사후세계였기에 단순한 형태를 띈다. 내가 죽으면 후대에 그릴 사람이 있어야 하기에 단수하게 만들었음
- 그리스
    - 종교적 숭배의 대상이면서도 동시에 미적 감상의 대상
    - 완벽한 비율, 비례를 가진 것이 신과 닮았다고 생각
    - 가장 아름다운(수학적 비례)cannon에 맞는 예술을 만들려 함
    - 눈에 보이는 형상의 극대화를 통해 이데아의 세계를 반영한다.
## 중세 예술
- 로마제국이 사라지고, 그리스로마신화가 끝난 뒤 기독교가 탄생
- 언어를 몰라서 그림으로 성경을 알려줌 -> 형식보다는 실질적, 아름다운 색 -> Material Definition
- 플라톤 동굴의 비유
    - 자신이 참이라 생각했던 것이 거짓(그림자), 바깥 세상에 나간 죄수는 잘 놀다가 다시 동굴에 와서 다른 죄수를 설득하려하지만 오히려 죄수를 죽임
    - 동굴은 현실(이데아에서 파생된 거짓), 바깥 세상은 이데아
- 플라톤의 상기설
    - 영혼과 현실의 세계를 가로지르는 레테 강이 있다.
    - 아기가 태어날 때 영혼의 세계에서 한 명이 발탁되어 현실로 가는데, 레테강에서 물을 마시고 기억을 망각함
    - 어느날 전시회에서 어떤 작품을 보는데 전율이 흐름 -> 원래 알고있었던 영혼의 세계와 접촉하고 감동받아 눈물을 흘리게 됨(상기) -> 기억을 찾기 위해 철학, 예술을 배움
    - => 추함과 아름다움이 명확하게 구분된다.
- 상기설에 대한 반박
    - 비율이 맞는데 우울한 사람이 아름다운가 아니면 비율이 안 맞는데 행복해하는 사람이 아름다운가? -> 플라톤이 대답을 못함
    - 하나님이 빛을 내려주는데(빛 유출설) 하나님의 빛을 받아내는 사람이 아름다운가, 빛을 못 받아내는 사람이 아름다운가? -> 수적 비례가 아니라 빛을 받아내는지가 아름다움의 요소가 된다.
        - 어떻게 빛을 받아낼 수 있을까? -> 스테인드글라스 ,촛불, 금은보화
        - 그림이 단순하고, 구성주의적으로 변함
- 중세예술과 현대예술
    - 중세 예술 : 가시적 세계 무시(초월적 내세 중시), 원근법 무지, 비가시적인 것의 가시화, 형식(재료)을 통한 내용 전달, 기하학적 묘사 > 추상
    - 현대 예술 : 가시적 세계 초월, 원근법 무시, 비가시적인 것의 가시화, 재료의 예술, 추상적 묘사 > 단순화
    - 울트라 마린 -> 코발트 블루(반 고흐)
    - 중세에는 예술가가 기능공이 되었다.
## 르네상스
- 십자군 전쟁으로 초상화를 많이 그리게 됨 -> 얼마나 닮게 그리는가가 중요 -> 그림을 보석으로 그리는 것이 천시됨 -> 사물을 묘사할 때 신의 창조 질서대로 묘사하라(토마스 아퀴나스)
- 수학자, 과학자가 예술가가 되었기에 엄청난 신분과 부를 가지게 됨
- 회화기법 : 유화, 원근법, 레이어
- 오네스티 가문의 레이어
    - 나스타지오가 피올라에게 고백했지만 피올라가 거절 -> 상심에 빠진 파스타치오는 숲을 걷다가 5월 금요일(비너스의 날) 쫓기고 있는 비너스 목격 -> 비너스를 쫓던 남자가 여자를 죽이고 심장을 들개에게 던짐 -> 알고보니 남자가 여자에게 고백했다가 거절당해서 죽인 후에 저주를 받아 계속 여자를 쫓게 됨 -> 나스타치오가 그 말을 듣고 연회를 열어서 파올라 초대 -> 여자를 쫓던 남자가 와서 난리침 -> 파올라가 저주에 걸릴까봐 결혼함
    - -> 원근법, 레이어 등을 볼 수 있음
- 르네상스 레오나르도 다빈치의 모나리자
    - 모나리자는 몰락한 귀족의 자녀. 돈을 위해 피렌체의 부호와 결혼 -> 피렌체의 부호가 모나리자를 그려달라고 함 -> 다빈치가 4년동안 못 그리니까 궁중 미술가에서 퇴출 -> 프랑스 부호가 초대해서 모나리자를 프랑스에 가서 그리려 함 
    - 예술 의지의 변화
        - 기하학/평면 배치 -> 시각피라미드/원근법/3차원 구성
        - 재료의 사용/빛의 사용으로 빛의 재현 -> 흑과 백의 조화/명암표현으로 빛의 재현

## 로코코 양식을 비판한 '요한 빙켈만'
- 고(옛 고) 양식(탄생) : 예술은 사람과 같은 유기체이다. -> 태어나면 성장하고 죽는다 -> 예술은 기존의 것을 모방하게 된다. -> 그리스가 이집트의 양식을 모방했다.
- "정면성의 원리"
    - 눈,코,입을 앞에서 본 것, 옆에서 본 것을 비교해보면 가장 잘 나타나는 것이 있음
    - 이러한 정면성의 원리가 그리스가 이집트 예술을 모방했다는 증거이다!
    - => 이집트의 정신을 완전히 벗어나지 못했다!
- 숭고의 양식(성장) : (광각운동)나의 지각 범위를 넘어서는 엄청난 물체가 나를 압도하는 느낌
- 미의 양식(완숙) : 드디어 그리스만의 스타일을 찾았다. -> 객관적 비례 + 우연적 요소 => 중신과 물질이 제대로 조화가 되었다.
- 모방의 양식(소멸) : 이렇게 모방이 계속되다 보면 물질이 정신을 압도한다.
    - 바로크와 로코코가 모방을 계속하다보니 보잘것 없어진다. -> 그리스 이후의 예술은 다 죽은 예술이다.

    - 모든 예술은 모방하는 것이고, 그렇다면 완벽한 아름다움을 모방하자(Canon)

## 헤겔의 절대정신
- 빌켈만 두둔
> "인간의 모든 역사, 예술은 물질로 외화된 정신이 다시 정신으로 돌아가는 과정이다"
- 플라톤의 망각의 강(레테 강) 이야기와 비슷함
- 하나님이 예수(물질)이 되었다가 청년이 되면 하나님의 아들이라는 것을 깨닫고, 다시 정신으로 돌아감
- 벽화, 그리스 조각상은 물성이 너무 강함(움직이지 못함) -> 캔버스 -> 사진기 -> 디지털 카메라(물성이란 존재하지 않음)
- 예술 발전의 3단계
    - 상징 예술 : 이집트, 메소포타미아 - 물질이 정신을 압도함 - 영혼이 애기일 때
    - 고전 예술 : 그리스 - 물질과 정신 공존 - 영혼이 자신의 정신을 알게 됨(이때가 예술의 정점)
    - 낭만 예술 : 르네상스 이후 - 회화 - 정신이 물성을 압도(하나님이 하늘로 돌아간 상태 - 예술은 죽었다)
- 독일의 작가가 그냥 돌로 구역을 만들었는데, 기자가 그것을 예술 작품으로 착각함
    - 과연 작가과 기자 중 누가 작가인가?
- 그리스(예술의 시대) -> 중세(종교의 시대) -> 낭만예술(철학의 시대)
- 절대정신
    - 정신은 절대자 -> 자연은 절대자(정신,관념)가 자신을 외화(물질)한 것 -> 자기인식
    - 정신 : 절대정신 -> 세계의 근원적 존재/이데아 -> 객관적 관념론
    - 의식 : 자기인식 -> 자연의 정신(로고스, 법칙)발견 -> 자연(물질>정신)의 재발견 -> 진정한 자기자신 끄집어 냄 -> 절대정신 도달
- 음악은 물질성 정도가 약함(소리, 진동) -> 정신성에 가까움
- 문학은 철학과 닮음 -> 물질성 정도가 약함
- => 예술은 낭만주의에 들어서면서 끝났다.
